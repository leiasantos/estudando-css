Seletores

No CSS, seletores são usados para "selecionar" ou "apontar" para os elementos HTML aos quais você deseja aplicar estilos. Existem vários tipos de seletores, cada um com uma finalidade específica.

  • id
  • class
  • type| element| tag
  • atributo
  • universal
